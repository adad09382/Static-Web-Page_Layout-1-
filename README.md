# Static-Web-Page_Exercise-1 
靜態網站練習1 (圍棋資訊網)

本網頁主幹以基礎html + css 指令完成，背景特殊動畫套用cdnjs 中的canvas-nest.js ，日本頁面圓餅圖套用引入google chart code

點擊導覽頁面可連結至其他頁面，導覽列會將停留頁面文字標黃，當游標移至導覽列文字時也會使文字標黃。
